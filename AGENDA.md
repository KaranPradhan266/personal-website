# Portfolio Website Agenda

## Goals
- Present a clear, compelling personal brand and story.
- Showcase selected projects with outcomes and impact.
- Provide easy contact paths for opportunities.

## Audience
- Recruiters and hiring managers
- Potential collaborators and clients
- Community peers

## Sections
1. Hero
   - Name, role, short value statement
   - Primary CTA (Contact / Resume)
2. About
   - Short bio, focus areas, tech stack highlights
3. Featured Work
   - 3-5 projects with role, problem, solution, impact
   - Links to live demo and source
4. Skills
   - Grouped by category (Frontend, Backend, Tools)
5. Experience
   - Timeline or cards, key achievements
6. Writing / Talks (optional)
   - Links to blog posts or talks
7. Contact
   - Email, social links, location/timezone
8. Footer
   - Copyright, quick links

## Content Checklist
- Professional headshot or illustration
- Up-to-date resume link
- Project visuals (screenshots/GIFs)
- Metrics (performance, scale, users, results)
- Testimonials (optional)

## Design + UX
- Centered, single-column editorial layout
- Content constrained to 640-720px max width
- Large negative space on both sides; no multi-column grids
- Calm, minimal, senior-engineer aesthetic
- Dark mode by default
- Strong typography hierarchy (clear h1, h2, body rhythm)
- Vertical stacking of sections (no cards, no masonry)
- No heavy gradients, no glassmorphism, no flashy effects
- Mobile-first layout
- Clear typography and spacing
- Strong visual hierarchy
- Fast loading and accessible

## Technical Tasks
- Set up routing and page sections
- Build reusable components
- Add SEO/meta tags and Open Graph
- Add analytics (optional)
- Deploy (Vercel or similar)

## Timeline
- Week 1: Content + wireframe
- Week 2: Build core sections
- Week 3: Polish, optimize, deploy
